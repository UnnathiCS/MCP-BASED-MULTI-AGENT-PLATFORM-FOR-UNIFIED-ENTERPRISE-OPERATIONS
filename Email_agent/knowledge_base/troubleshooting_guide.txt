NexusFlow Troubleshooting Guide

Common Issues and Solutions:

1. Authentication Errors

Issue: "401 Unauthorized" error when using API
Solutions:
- Verify API key is correct and not expired
- Check API key has required permissions (Read/Write/Execute)
- Ensure API key is included in Authorization header: "Bearer YOUR_API_KEY"
- Regenerate API key if unsure: Settings > API Keys > Generate New Key
- Check if account is active and not suspended

Issue: "403 Forbidden" error
Solutions:
- API key may not have required permissions
- Check workflow ownership - you can only access workflows you created or are shared with you
- Verify account tier allows the requested operation
- Contact support if issue persists

2. Workflow Execution Issues

Issue: Workflow stuck in "Processing" state
Solutions:
- Check execution logs: Workflows > [Workflow Name] > Executions > [Execution ID] > Logs
- Look for timeout errors or external service failures
- Verify all connectors are properly configured
- Check if external services (Salesforce, Zendesk, etc.) are accessible
- Retry execution: Executions > [Execution ID] > Retry
- If persistent, contact support with Workflow ID and Execution ID

Issue: Workflow execution fails immediately
Solutions:
- Check workflow configuration for errors
- Verify all required fields are filled
- Test individual connectors: Connectors > [Connector Name] > Test Connection
- Review error message in execution logs
- Ensure workflow is not paused: Workflows > [Workflow Name] > Settings > Status

Issue: "Rate limit exceeded" error
Solutions:
- Check your account tier limits
- Reduce execution frequency
- Upgrade to higher tier if needed
- Wait for rate limit window to reset (typically 1 hour)
- Contact support for Enterprise rate limit increase

3. Connector Issues

Issue: Connector connection fails
Solutions:
- Verify credentials are correct
- Check if external service is accessible
- Test connection: Connectors > [Connector Name] > Test
- Review connector-specific requirements in documentation
- Some connectors require IP whitelisting - check connector settings
- OAuth connectors may need re-authentication

Issue: Data not syncing from connector
Solutions:
- Check connector status: Connectors > [Connector Name] > Status
- Verify connector permissions in external service
- Review sync schedule settings
- Check for errors in connector logs
- Manually trigger sync: Connectors > [Connector Name] > Sync Now

4. Performance Issues

Issue: Dashboard loading slowly
Solutions:
- Clear browser cache and cookies
- Try different browser or incognito mode
- Check internet connection speed
- Reduce date range in analytics queries
- Complex dashboards may take 10-15 seconds on first load
- Contact support if consistently slow

Issue: Workflow execution taking too long
Solutions:
- Review workflow complexity - simplify if possible
- Check external service response times
- Review execution logs for bottlenecks
- Consider breaking complex workflows into smaller workflows
- Check account tier execution limits
- Upgrade to Enterprise for dedicated resources

5. Data Issues

Issue: Missing data in workflows
Solutions:
- Check data source connectors are working
- Verify data filters are not too restrictive
- Review workflow logic for data transformation errors
- Check execution logs for data processing errors
- Verify data retention settings haven't deleted old data

Issue: Incorrect data transformations
Solutions:
- Review workflow mapping rules
- Test individual transformation steps
- Check data types match expected formats
- Review execution logs for transformation errors
- Verify source data format matches expectations

6. Billing and Account Issues

Issue: Cannot access account after payment
Solutions:
- Check payment status: Settings > Billing > Payment History
- Verify payment method is valid
- Check for failed payment notifications
- Contact billing support: billing@companyz.com
- Account may be suspended due to payment failure

Issue: Feature not available
Solutions:
- Check account tier - some features require Professional or Enterprise
- Verify feature is enabled: Settings > Features
- Review account limits
- Upgrade account if needed
- Contact support to verify feature availability

7. Integration Issues

Issue: Webhook not receiving events
Solutions:
- Verify webhook URL is accessible and returns 200 status
- Check webhook configuration: Settings > Webhooks
- Review webhook logs for delivery attempts
- Ensure webhook endpoint is not blocking requests
- Test webhook manually: Webhooks > [Webhook Name] > Test

Issue: Custom integration not working
Solutions:
- Review API documentation for correct endpoints
- Verify API key has required permissions
- Check request format matches API specification
- Review error responses for specific issues
- Test with API testing tool (Postman, curl)
- Contact developer support for custom integrations

Getting Help:

Support Channels:
- Email: support@companyz.com
- Response time: 24 hours (Professional), 48 hours (Free)
- Phone: Available for Enterprise customers
- Documentation: https://docs.nexusflow.com
- Community Forum: https://community.nexusflow.com

When Contacting Support:
- Include Workflow ID or Execution ID if applicable
- Describe the issue in detail
- Include error messages
- Attach relevant logs or screenshots
- Mention steps you've already tried

- Critical production issues: support@companyz.com with "URGENT" in subject
- SLA: 1-hour response for Enterprise critical issues
