================================================================================
TROUBLESHOOTING
================================================================================

"Agent is offline" 🔴
→ Make sure you ran it in Terminal 1 or 2
→ Wait for "Application startup complete"

"Cannot connect to port 8000"
→ Another process using it
→ Try: lsof -i :8000
→ Kill it: kill -9 <PID>
→ Restart agent

"Streamlit not found"
→ pip install streamlit

All agents must be running simultaneously!
